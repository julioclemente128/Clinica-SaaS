# Clínica SaaS — Contexto del Proyecto

## Qué es esto
Micro-SaaS para clínicas estéticas en Chile y LATAM.
Resuelve el problema de clínicas que invierten en Meta Ads, capturan leads y los pierden por falta de seguimiento automático.

## Producto
- Landing page personalizada por clínica
- Formulario de captura de leads (nombre, email, teléfono, servicio)
- Secuencia automática de 3 emails generada con Claude API
- Dashboard simple para ver leads capturados

## Cliente demo
Clínica Lumina — tratamientos faciales — Las Condes, Santiago.

## Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js + Express
- Base de datos: Supabase
- Email automation: Brevo
- IA: Claude API
- Deploy: Vercel (frontend) + Railway (backend)

## Estado actual

## Principios
- Código simple y legible, sin frameworks por ahora
- Cada cambio debe ser explicado para que el desarrollador entienda qué hace y por qué
- Mobile responsive desde el inicio
